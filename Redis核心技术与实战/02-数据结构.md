#### 压缩列表

- 类似数组
- 连续内存空间
- ![img](https://static001.geekbang.org/resource/image/49/b5/49fd8d46eb94f463ace98717f11c2cb5.jpg)

压缩的目的是用来降低空间损耗。

这里的降低是相较于传统的数组而言。

传统的数组的每个元素的长度是一样的。以上图为例，我们每个存储单元的大小应该是最大的元素的长度。



#### 双向链表

```java
class ListNode{
    ListNode prev;
    ListNode next;
    Object value;
}

class List {
    ListNode head;
    ListNode tail;
    long len;
}
```
